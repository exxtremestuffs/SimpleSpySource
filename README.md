# SimpleSpy

[![](https://dcbadge.vercel.app/api/server/JbjVd6PMvz?style=plastic)](https://discord.gg/JbjVd6PMvz)

A maintained fork of SimpleSpy.

## Features

* Event logging
* Exclude & block events
* Generate code from previous events (includes a serializer)
